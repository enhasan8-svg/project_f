System Architecture

This graduation project focuses on detecting AI-generated images using advanced computer vision forensics.

Capsule Networks

Utilizes dynamic routing to preserve spatial relationships within images, effectively detecting subtle structural inconsistencies.

VGG Extractor

Pre-trained VGG-19 layers extract high-level semantic features before passing them to the primary capsule layers.

Grad-CAM

Provides visual transparency by highlighting the regions that most strongly influenced the final deepfake prediction.

Pixel Forensics

Future modules will include Error Level Analysis to detect pixel tampering and artificial compression artifacts.

Metadata Analysis

Future expansions will automatically extract and review hidden EXIF data and generation tags from suspect files.

Purpose

Designed as a high-fidelity academic project to demonstrate state-of-the-art methods in distinguishing authentic media.
